Технологическое задание для бота в телеграмм

Необходимо написать код для бота в телеграмм, который будет помогать репетитору по химии.
Для айди бота и остальных кредов создай отдельный файл

Функции:
1) Администратор
1.1) Для администратора у бота должны быть отдельные кнопки:
 - Добавить ученика: Администратор направляет tg username ученика, который получает доступ к боту, а также информацию о днях и времени когда он занимается. 
 - Информация о ученике: Выводится список доступных лекций и расписание занятий
 - Редактировать ученика: выводится список всех учеников, после выбора нужного можно добавить лекцию к просмотру, или отредактировать расписание или удалить ученика
 - Добавить лекцию: Администратор направляет файл лекции а также ее название, лекция добавляется в базу данных (их будет до 50 штук)
 - Удалить лекцию: Необходимо вывести таблицу с номерами и названиями лекций, администратор вводит номер лекции и она удаляется из Базы данных

 2) Ученик
 2.1) Для ученика должны быть доступны кнопки:
 - Просмотреть расписание
 - Просмотреть лекции, где выводится список всех доступных лекций, после выбора нужной из бд возвращается файл нужной лекции

 3) Необходимо реализовать легковесную файловую базу данных (Redis)
 - Для хранения учеников и айди администратора также разарешается использовать бд или конфигурационный файл (json)

 4) Сначала бот будет развернут локально для тестирования, а затем будет развернут на сервере.
 Сначала реализовать локальную версию.
